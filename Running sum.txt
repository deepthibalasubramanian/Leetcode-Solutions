Problem:

Given an array nums. We define a running sum of an array as 
runningSum[i] = sum(nums[0]â€¦nums[i]).

Return the running sum of nums.

Solution:

class Solution:
    def runningSum(self, nums: List[int]) -> List[int]:
        length=len(nums)
        for i in range(1,length):
            nums[i]+=nums[i-1]
        return nums
